# MFG-AI: Neural Industrial Interface 🏭🚀

**MFG-AI** is a high-performance MERN-stack dashboard designed to bridge the gap between human intuition and industrial production. Developed during the **Nova Nexus 2026 Hackathon**, it transforms raw natural language into structured manufacturing actions.

---

## 🌟 Unique Selling Point (USP)
**"The Neural Interface"** Traditional ERP systems are bogged down by complex forms and manual data entry. MFG-AI utilizes a **Natural Language to Action (NL2A)** pipeline. Floor managers can simply type or speak their requirements, and our system instantly parses, validates, and visualizes the production data.

## 🛠 Tech Stack
- **Frontend:** React.js (Vite) with a high-contrast Glassmorphism UI.
- **Backend:** Node.js & Express.js.
- **AI Engine:** **Groq API** (Llama 3) for ultra-low latency intent extraction.
- **Database:** MongoDB Atlas (NoSQL).
- **Security:** JWT-based user isolation and Bcrypt password hashing.

## 🔄 Workflow
1. **Command:** User inputs a natural language instruction (e.g., *"Need 500 Steel Bolts by June 12"*).
2. **Inference:** The **Groq LPU** parses the text into a structured JSON object in milliseconds.
3. **Validation:** The backend checks the schema and assigns the order to the specific **User ID**.
4. **Visualization:** The **Production Pipeline Heatmap** and **Live Feed** update in real-time to reflect the new state of the factory floor.

## 🚀 Key Features
- **Zero-Trust Isolation:** Every user has a strictly isolated data silo.
- **Live Heatmap:** A bird’s-eye view of production health (Received vs. Processing vs. Accepted).
- **Millisecond Latency:** Powered by Groq for near-instant response times.
- **Mobile Responsive:** Designed for tablets and mobile devices used on factory floors.
